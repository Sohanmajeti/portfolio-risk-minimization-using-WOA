# Portfolio Risk Minimization using Whale Optimization Algorithm

This project uses the Whale Optimization Algorithm (WOA) to optimize stock portfolio allocation using real market data.

## Features
- Risk minimization
- Portfolio weight optimization
- Data visualization
- Python implementation

## Tools Used
Python, NumPy, Pandas, Matplotlib, yfinance
